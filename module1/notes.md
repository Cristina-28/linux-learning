Linux is the OS of servers and the cloud.
pwd shows your current location.
ls lists files in directory.
cd moves between directories.
mkdir creates a new folder.
cat reads the whole file.
head only reads the first N lines (head -n N folder/file.ext)
tail only reads the last N lines (like head)
nano is a basic text editor. ctrl+o to save, ctrl+x to exit
<<BE CAREFUL>> when using echo, >> appends to the text file, > OVERWRITES it
